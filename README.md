# rust-task
A task management program written while learning Rust

## Usage
Creating new tasks:

`$ rust-task new --name <name> --class <class> --datedue <date>`

Listing tasks:

`$ rust-task list`

Removing tasks:

`$ rust-task rm --name <task to remove>`

## Building and running

Building:

`$ cargo build`

Running:

`$ cargo run`
