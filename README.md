# rust-task
A task management program written while learning Rust

## Usage
Adding new tasks:

`$ rust-task add <name> <description>`

Listing tasks:

`$ rust-task ls`

Removing tasks:

`$ rust-task rm <name>`

Editing tasks:

`$ rust-task edit <name>`

## Building and running

Building:

`$ cargo build`

Running:

`$ cargo run`
